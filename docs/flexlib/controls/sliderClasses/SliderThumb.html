Classpublic class SliderThumb
InheritanceSliderThumb mx.controls.Button

The SliderThumb class represents a thumb of a Slider control. The SliderThumb class can only be used within the context of a Slider control. You can create a subclass of the SliderThumb class, and use it with a Slider control by setting the sliderThumbClass property of the Slider control to your subclass.
